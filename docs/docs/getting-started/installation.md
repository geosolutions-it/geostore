## Prerequisites

- Java 11 or later
- Apache Maven 3.6+
- A servlet container (e.g., Apache Tomcat 9+)

## Building from Source

```bash
git clone https://github.com/geosolutionsgroup/geostore.git
cd geostore
mvn clean install
```

## WAR Deployment

Deploy the generated WAR file from `src/web/app/target/geostore.war` to your servlet container.
